Tabela de Dados Filtrável e Ordenável:
Desafio:
Crie uma tabela de dados que permita aos usuários filtrar os dados com base em critérios específicos e ordenar as colunas de forma ascendente e descendente.
Requisitos:
Inputs para filtrar dados por diferentes critérios.
Capacidade de ordenar as colunas clicando nos cabeçalhos.
Atualização dinâmica da tabela no DOM conforme os filtros e ordenações são aplicados.
Utilize eventos do DOM (como `click`, `input`, `change`) para aplicar filtros e ordenações.
