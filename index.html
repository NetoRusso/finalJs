Javascript {JS}

By Neto Russo ©

Um mundo de possibilidades!

Durante o Módulo de Javascript do Bootcamp de Java Fullstack da Soul Code com parceria Sankhya , o professor Adriano Gomes abordou muitos assuntos, que vamos falar a baixo, e todos esses tópicos foram aplicados nos exercícios finais aqui demonstrado!


O Javascript é uma linguagem de programação fundamental para a web moderna.
Ele permite que páginas web interajam com os usuários, respondam a eventos, manipulem dados e criem experiências dinâmicas e envolventes. Prepare-se para descobrir um mundo de possibilidades!

Unidade I: Os Fundamentos do Javascript

Começamos explorando os conceitos básicos do Javascript, como variáveis, tipos de dados, operadores e estruturas de controle. Essa base sólida é fundamental para dominar as funcionalidades mais avançadas.

Unidade II: Decisões e Repetições

Aprendemos a construir lógica de programação com estruturas de condição (if-else, switch-case) e de repetição (loops) para que seus programas tomem decisões e executem tarefas repetitivas de forma eficiente.

Unidade III: Funções: Blocos de Reutilização

Dominamos as funções no Javascript é essencial para organizar seu código, reutilizar trechos de código e melhorar a eficiência. Exploramos os conceitos de parâmetros, retorno e escopo para construir funções eficazes.

Unidade IV: Arrays e Strings: Manipulando Dados

Arrays e strings são ferramentas essenciais para armazenar e manipular dados textuais e sequencias de valores. Aprendemos a acessar, modificar, percorrer e organizar esses dados com métodos poderosos do Javascript.

Unidade V: Objetos: Modelagem de Dados Complexos

Objetos são estruturas flexíveis para representar dados complexos, como pessoas, produtos ou configurações. Descubrimos como criar, acessar, modificar e organizar objetos com propriedades e métodos.

Unidade VI: DOM: Controlando a Interface da Web

DOM (Document Object Model) é a interface que permite interagir com os elementos HTML de uma página. Aprendemos a selecionar, manipular, adicionar e remover elementos, estilos e eventos, transformando suas páginas web em experiências interativas.
